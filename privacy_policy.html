BetBlock — Política de Privacidade

Última atualização: Junho de 2026

1. Dados coletados

O BetBlock não coleta, armazena, transmite nem compartilha nenhum dado pessoal dos usuários.

A extensão armazena localmente no seu navegador apenas:

  • Preferência de ativação (ligado/desligado) — via chrome.storage.local
  • Contador de anúncios bloqueados na sessão — via chrome.storage.local

Esses dados ficam exclusivamente no seu dispositivo e nunca são enviados a nenhum servidor externo.

2. Permissões utilizadas

  • declarativeNetRequest: bloqueia requisições de rede para domínios de apostas conhecidos antes de carregar.
  • storage: salva preferências locais no navegador.
  • tabs: lê o ID da aba ativa para exibir a contagem por página no popup.
  • host_permissions (<all_urls>): necessário para injetar o script de detecção em todas as páginas e identificar anúncios de apostas no conteúdo da página.

Nenhuma dessas permissões é utilizada para rastrear, identificar ou transmitir dados sobre o usuário ou sobre os sites visitados.

3. Serviços de terceiros

O BetBlock não utiliza nenhum serviço de terceiros, SDK de analytics, telemetria ou sistema de relatório de erros.

4. Menores de idade

Esta extensão não coleta dados de nenhum usuário, incluindo menores de idade.

5. Alterações nesta política

Qualquer alteração será publicada nesta página com atualização da data.

6. Contato
